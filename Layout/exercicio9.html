Bem-vindo à Página Dinâmica!

Redimensione a janela do seu navegador para ver como a cor de fundo e os tamanhos dos textos mudam suavemente. Utilizamos CSS transitions e media queries para alcançar este efeito.

Observe como as transições aplicadas às propriedades background-color, font-size, e color criam uma experiência visual mais agradável quando o layout se adapta a diferentes tamanhos de tela.
