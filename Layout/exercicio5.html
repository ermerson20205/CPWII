Cabeçalho do Site

Exercício 1

Padrão em F: Para páginas com muito texto (blogs, notícias). O olhar foca no topo e na lateral esquerda. Padrão em Z: Para páginas com pouco texto (landing pages, logins). O olhar segue um traçado em "Z" para levar à conversão. Regra de ouro: Use F para leitura e Z para ação (clique/venda).

Exercício 2

O Web Design Responsivo é a técnica que faz um site se adaptar automaticamente a qualquer tamanho de tela (celular, tablet ou PC) usando um único código. Como funciona: Usa grades fluídas (porcentagens), imagens flexíveis e regras de código (media queries) para reorganizar os elementos conforme o espaço disponível.

Exercício 3

Layout Líquido (Fluído): Usa porcentagens. Vantagem: Adapta-se a qualquer monitor e aproveita todo o espaço da tela. Desvantagem: Pode esticar demais o texto em telas gigantes, dificultando a leitura. Layout Fixo: Usa pixels. Vantagem: O design fica exatamente igual em qualquer lugar (maior controle visual). Desvantagem: Péssimo para celulares, pois cria barras de rolagem laterais ou exige zoom.

Exercício 4

O que faz: Define que a largura do site deve ser igual à largura da tela do aparelho (width=device-width) e ajusta o zoom inicial para 100%. Importância: Sem ela, o celular tenta exibir a versão de desktop "espremida", deixando letras e botões minúsculos.
